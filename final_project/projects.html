Stellar Body Motion Simulator

This is an iOS application whose primary function is a simulator of stellar body motion in 3D space due to gravity. For the time being, the stars are set to identical. A user can rotate the view angle and add new celestial bodies into the simulator, or the user can go to the auto-replay simulator which automatically restarts a random simulation whenever it detects an ejected star.

The project is not deployed onto app store and its feature is still being enrished, but you could download the source code and run it locally.
